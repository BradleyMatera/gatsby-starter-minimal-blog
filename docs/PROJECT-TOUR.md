
# Project tour

## Key folders
- `src/pages/` - Gatsby pages (store, purchases, support).
- `netlify/functions/` - backend endpoints.
- `netlify/db/migrations/` - schema migrations.
- `netlify/db/seed/` - seed data.
- `static/store/` - store product images (SVG placeholders).
- `content/pages/` - MDX pages.
- `content/posts/` - blog posts.
