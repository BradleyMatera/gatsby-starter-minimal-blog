  <h2 class="content-heading-2">JavaScript Example</h2>
  <p class="content-paragraph">
    Here's a simple JavaScript function to calculate the factorial of a number:
  </p>
  <pre><code>function factorial(n) {
  if (n === 0 || n === 1) {
    return 1;
  }
  return n * factorial(n - 1);
}

console.log(factorial(5)); // Output: 120</code></pre>
  
  <h2 class="content-heading-2">Python Example</h2>
  <p class="content-paragraph">
    Below is a Python script for a basic calculator:
  </p>
  <pre><code>def calculator(a, b, operation):
    if operation == 'add':
        return a + b
    elif operation == 'subtract':
        return a - b
    elif operation == 'multiply':
        return a * b
    elif operation == 'divide':
        return a / b if b != 0 else "Cannot divide by zero"
    else:
        return "Invalid operation"

print(calculator(10, 5, 'multiply'))  # Output: 50</code></pre>
